O que é HTML

HTML (HyperText Markup Language) é a linguagem usada para criar páginas na web. Ela organiza o conteúdo por meio de tags, que definem elementos como textos, imagens, links e outros componentes, permitindo que navegadores exibam as informações de forma estruturada.

O que é CSS

CSS (Cascading Style Sheets) é a linguagem usada para estilizar páginas da web. Ele define como os elementos HTML devem ser exibidos, permitindo controlar cores, fontes, layouts e a aparência geral de uma página. Com CSS, é possível deixar um site mais visualmente atraente e organizado, separando o conteúdo da apresentação.

O que é JavaScript

JavaScript é uma linguagem de programação usada para criar funcionalidades dinâmicas em páginas da web. Diferente do HTML e do CSS, que estruturam e estilizam o conteúdo, o JavaScript permite que o site responda a interações do usuário, como cliques, animações, validação de formulários e a atualização de conteúdo sem recarregar a página. Ele torna os sites mais interativos e dinâmicos.

O que é Java

Java é uma linguagem de programação de propósito geral, orientada a objetos, e bastante popular por sua versatilidade e robustez. Ela é usada para desenvolver uma ampla gama de aplicações, como softwares corporativos, aplicativos móveis (especialmente para Android), sistemas web, e até mesmo sistemas embarcados. Java é conhecida por seu princípio "escreva uma vez, execute em qualquer lugar" (WORA), o que significa que o código escrito em Java pode ser executado em diferentes plataformas sem precisar ser modificado, desde que a máquina tenha a Java Virtual Machine (JVM) instalada.

O que é Github

GitHub é uma plataforma online que permite o armazenamento e o gerenciamento de código-fonte usando o sistema de controle de versões Git. Ele é amplamente utilizado por desenvolvedores para colaborar em projetos, compartilhar código e acompanhar mudanças ao longo do tempo. Além disso, o GitHub facilita a criação de repositórios (locais onde o código é armazenado), o rastreamento de issues (problemas ou tarefas), e a realização de pull requests (solicitações para fundir alterações no código). É uma ferramenta essencial para projetos de software, especialmente no trabalho em equipe e no desenvolvimento de código aberto.

O que é git

Git é um sistema de controle de versão distribuído, usado para rastrear mudanças em arquivos e colaborar no desenvolvimento de software. Ele permite que vários desenvolvedores trabalhem simultaneamente em um projeto sem sobrescrever o trabalho uns dos outros, mantendo um histórico detalhado de todas as alterações feitas no código. Com o Git, é possível criar diferentes "ramificações" (branches) do projeto para desenvolver novos recursos ou corrigir bugs, e depois mesclar essas mudanças no projeto principal. Em resumo, o Git ajuda a organizar e gerenciar projetos de software, garantindo que todas as versões e modificações sejam controladas de maneira eficiente e segura.
